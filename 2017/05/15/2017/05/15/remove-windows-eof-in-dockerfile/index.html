换行符

Windows 上的换行符是 \r\n;

*nix 系统中使用 \n。

如果在 Windows 上修改，甚至通过 Git 等版本控制工具移动文本文件，都会导致换行符的改变。

如果换行符变更，很多脚本将会运行报错，如果在 Linux 系统中使用 vi 编辑器查看，会每行后面看到 ^M 字符。
